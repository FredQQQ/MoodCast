# MoodCast 精神气象台

一个由卡卡卡卡皮巴拉制作的趣味精神天气观测网页。

## 功能

- 随机情绪天气测试与分享卡片
- 好友双人相处气象和气象物资
- 本机天气历史与天气图鉴
- 今日大众精神天气、同类天气人数与真实稀有度
- 按时段汇总的公共气象简报与简报分享图
- 匿名结果准确度反馈

## 本地打开

静态功能可以直接打开 `index.html`。后台功能需要通过 Netlify 部署，或安装 Netlify CLI 后使用本地开发服务器。

## 数据说明

后台只记录匿名随机设备编号、天气类型、测试模式、观测时段和可选准确度反馈，不上传具体题目答案、姓名或位置。

详细公共统计位于站内第二层“今日气象简报”页面：首页和测试结果页都可以进入。新版本上线后产生的观测会参与时段统计；旧版观测仍会计入当天总数，但无法还原具体时段。

## Netlify

项目使用：

- Netlify Functions 处理匿名请求
- Netlify Blobs 保存观测和反馈
- `netlify.toml` 配置构建、函数和 API 路由

连接 GitHub 后，推送到生产分支会自动触发部署。
